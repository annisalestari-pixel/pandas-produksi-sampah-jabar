>> Pengolahan Data Produksi Sampah Jawa Barat <<

# Deskripsi
Project ini bertujuan untuk mengolah dan menganalisis data jumlah sampah
berdasarkan Kabupaten/Kota di Provinsi Jawa Barat.
Pengolahan data dilakukan menggunakan bahasa pemrograman Python
dengan bantuan library Pandas.

# Dataset
Dataset yang digunakan bersumber dari Open Data Jawa Barat,
yaitu data "Jumlah Sampah yang Ditangani Berdasarkan Kabupaten/Kota di Jawa Barat".
Data ini memuat informasi nama kabupaten/kota, jumlah sampah, satuan, dan tahun.

# Tahapan Pengerjaan
Tahapan pengerjaan dalam project ini meliputi:
1. Membaca dataset dari file CSV menggunakan Pandas.
2. Memilih kolom data yang dibutuhkan, yaitu Kabupaten/Kota, jumlah sampah, dan tahun.
3. Menghitung total jumlah sampah pada tahun tertentu menggunakan metode `iterrows`.
4. Menghitung total jumlah sampah berdasarkan tahun.
5. Menghitung total jumlah sampah berdasarkan Kabupaten/Kota pada setiap tahun.
6. Menyimpan hasil pengolahan data ke dalam file CSV dan Excel.
